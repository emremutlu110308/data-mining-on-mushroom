Decision Tree ReadMe:
1. Import the project with Eclipse
2. The testing Java JDK version is 1.7.
3. The jxl.jar file should be imported in order to read the data.xls file for dataset.
4. Please reset the jar library path for jxl.jar if needed.
5. Run the Algorithm.java file for the results, which takes about 6-7 mins.
6. Check the Result.txt file for results of decision tree.
